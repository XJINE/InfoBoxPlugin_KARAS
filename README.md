# Infobox_KARASPlugin

This is pulugin for KARAS-php ver.
You can make infobox like following image.

![sample](./sample.png)

## How to Use

```
[[infobox::
your text
]]
```

or

```
[[infobox::note::
your text
]]
```

There is some valid options, "note", "help", "caution", "warning", and "download".
